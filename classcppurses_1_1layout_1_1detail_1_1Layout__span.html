Detailed Description

template<typename Get_policy_t>
class cppurses::layout::detail::Layout_span< Get_policy_t >

Container view to iterate over a Widget's children, yielding layout info.

Shrinks on each iteration by dropping elements that have reached some limit. Get_policy_t: A functor type <Size_policy const&(Widget const&)> Used to retrieve a Size_policy to work with.
