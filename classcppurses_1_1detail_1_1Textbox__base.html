Detailed Description

Implements cursor movement on top of a Text_display, for use by Textbox.

void cppurses::detail::Textbox_base::cursor_down ( std::size_t  n = 1)

Move the cursor down n lines, scroll if at the bottom line.

Cursor is moved to the right-most Glyph if the moved to line does not extend as far as the current cursor column position.
