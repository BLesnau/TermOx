Detailed Description

template<typename Number_t = double>
class ox::Color_graph< Number_t >

Bounded box that can display added Color Coordinates within the Boundary.

X axis is horizontal and increasing from left to right. Y axis is vertical and increasing from bottom to top. Uses half-block characters, allowing two points per terminal cell.
