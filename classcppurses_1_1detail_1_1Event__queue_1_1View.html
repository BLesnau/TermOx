Detailed Description

template<Event::Type filter_type>
class cppurses::detail::Event_queue::View< filter_type >

Provides iterator access to filter_ type elements in an Event_queue.

filter: None - gives you all event types except Paint and Delete. This type is for exclusive use by Event_engine class, single thread.
