Detailed Description

Text input box with input validator and Signal emitted on Enter Key press.

Initial text is in ghost color and cleared on initial focus. Height is fixed at 1.

void cppurses::Line_edit::set_validator ( std::function< bool(char)>  validator)

Set the input validator, allowing or disallowing specific char types.

The default validator always returns true, allowing all chars. Invalid character input will result in no input being given to the Line_edit.
