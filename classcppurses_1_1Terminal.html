◆ raw_mode()

void cppurses::Terminal::raw_mode ( bool  enable = true)

Enable or disable raw mode.

In raw mode, the interrupt, quit, suspend, and flow control characters are all passed through uninterpreted, instead of generating a signal. This mode is off by default.

Set the rate at which the screen will update.

User input will immediately flush, but other event loops are only processed every refresh rate. Default is 33ms.
