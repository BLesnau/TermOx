Detailed Description

loop_function() waits for a given time interval, then returns.

Not useful on its own, but can be inherited from, and its loop_function can be called on from the derived classes loop_function to perform the wait. Accounts for time since last wait so interval is not dependent on calculations in between the calls to wait.
